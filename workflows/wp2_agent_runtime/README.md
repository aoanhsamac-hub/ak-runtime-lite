# AK WP2 Agent Runtime Workflow

This workflow verifies the AK Agent Runtime Framework in dry-run mode.

Checks:

- Required files exist.
- Seven agents boot.
- Role boundaries are present.
- Agent memory access uses interface/client only.
- OpenCode is restricted to Lang Lieu.
- No direct execution is enabled.
- No secret access is allowed.
- Tests pass.
- Report exists.

## Cấp quyền thay đổi

### Cấp 0 — Không cần review lớn

- sửa typo tài liệu
- thêm ghi chú
- tạo report
- phân tích log
- tạo checklist

- đổi cấu trúc thư mục
- đổi naming
- thêm tài liệu mới
- thêm workflow

### Cấp 2 — Cần Lang Liêu review

- sửa code không liên quan execution
- refactor module
- thêm test
- sửa dashboard

### Cấp 3 — Cần Sage review

- risk kernel
- execution logic
- MT5 order logic
- portfolio
- live config
- emergency stop

### Cấp 4 — Cần Hùng Vương phê duyệt

- sửa Hiến pháp

- thay đổi quyền agent
- deploy live lớn
- tăng risk/chi tiêu tổng thể

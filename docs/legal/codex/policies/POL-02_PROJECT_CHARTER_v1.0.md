# Alkasik Kingdom Project Charter

## Identity

- Legal name: Alkasik Kingdom
- Technical name: AK
- Sovereign authority: Hung Vuong

## Objective

Alkasik Kingdom builds an independent system intended to replace Alkasik Legacy only after AK passes the Replacement Gate.

## Non-Legacy Runtime Rule

- AK must not import old runtime code from Alkasik Legacy.
- AK must not runtime-link to Alkasik Legacy.
- AK must not depend on Legacy runtime paths, databases, dashboards, API gateways, config loaders, bot scripts, trading scripts, or connector code.
- Legacy may be used only as archived knowledge, never as active runtime.

## Allowed Knowledge Inheritance

AK may inherit only approved legal documents, governance material, reports, lessons, approved knowledge, and approved datasets.

## Economic Scope

Trading is one economic sector inside Alkasik Kingdom. Trading is not the whole AK system.

## Operating Principles

- Governance Before Execution.
- Safety First.
- Backup First.
- Archive First, Delete by Policy.
- Observe-only default dashboard.
- Execution must fail-closed.

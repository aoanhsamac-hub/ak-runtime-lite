## Key Requirements (from WP3.5 doctrine and available documentation)

### Agent Governance
- All agents must follow Governance Before Execution
- Proposal ≠ Execution ≠ Authorization
- Knowledge ≠ Permission ≠ Capability
- Capability ≠ Power

### Agent Roles
- Janus: Coordination, planning, approval routing
- Sage: Risk review, governance gate control
- Lang Lieu: Technical implementation, code review
- Hermes: Memory, lesson, skill management
- Iris: Market intelligence, budget management
- Helen: Civilization intelligence, information review
- Yet Kieu: Security, runtime monitoring

### Agent Restrictions
- No autonomous execution without approval
- No risk kernel modification without Sage review
- No production deploy without Janus approval

# Alkasik Kingdom Sovereign Repository

This directory contains the legal, governance, decree, and sovereign corpus records for Alkasik Kingdom.

Technical runtime, connectors, Docker, environment files, and execution code remain under AK technical directories and are not governed by this directory layout.

## Structure

- `constitution/`: Constitution documents.
- `state_corpus/`: State Corpus documents.
- `laws/`: Active laws grouped by domain.
- `decrees/`: Active decrees grouped by domain.
- `quarantine/`: Legal/governance documents requiring review before activation.
- `legal_index.yaml`: Canonical index of active sovereign documents.
